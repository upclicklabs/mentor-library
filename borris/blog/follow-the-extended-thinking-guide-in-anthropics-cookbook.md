# Follow the extended thinking guide in Anthropic's Cookbook

## Overview

The extended_thinking directory within the Anthropic Cookbook repository provides hands-on examples for implementing extended thinking features in Claude.

## Contents

The extended_thinking directory contains two Jupyter notebooks:

1. **extended_thinking.ipynb** - Core extended thinking examples
2. **extended_thinking_with_tool_use.ipynb** - Extended thinking combined with tool use

## About Anthropic Cookbooks

This is Anthropic's official repository containing practical examples and guides for working with Claude AI models, including:

- Capabilities demonstrations
- Coding examples
- Tool use patterns
- Multimodal examples
- Fine-tuning guides

The notebooks provide educational resources demonstrating how to leverage Claude's extended thinking capabilities, both standalone and in combination with tool use.
