# Project Permissions

## Allowed Actions (no confirmation needed)

- Run `git pull`, `git add`, `git commit`, `git push` on this repo
- Run `curl` commands to the Notion API (api.notion.com)
- Run `curl` to noembed.com for video metadata
- Run `python3` with `youtube-transcript-api` for transcript extraction
- Run `mkdir -p` to create mentor content directories
- Read `.env` files for API tokens
- Create and edit `.md` files in this repo
- Use WebFetch to extract blog/article content

## Requires User Approval

- Adding URLs to Notion (`/research-mentor` must present results and wait for "approve")
