# Shopify Search & Discovery App

**Source:** Shopify Help Center

---

## Overview

The Shopify Search & Discovery app gives merchants more control over how customers discover products online. It includes settings and features to customize the store search experience, set up product search filters, and configure product recommendations.

---

## Key Features

### Search Customization

- Adjust the type of results returned in search results
- Assign specific search terms to products to feature them in search results
- Create synonym groups to ensure different words with the same meaning return the same results
- Modify search with the app for more targeted discovery

### Product Filtering

- Create filters for your online store that let customers refine which products are displayed on collection pages and in search results
- Manage filter behavior per collection and search context

### Product Recommendations

- Customize product recommendations shown throughout the store
- Use the app to set up and manage personalized recommendation sections
- Access detailed reports on recommendation performance

### Analytics and Reporting

Access a set of reports to determine how Discovery surfaces are performing and where to prioritize customization efforts:
- Assess how product recommendations are performing
- Identify search terms with no results
- Improve performance by adjusting recommendations and search behavior

### Predictive Search (Autocomplete)

Customize predictive search functionality - the autocomplete results that appear as customers type in the search bar.

---

## Sub-Features

- **Modifying search** - Adjust result type and boost specific products
- **Adding filters** - Create collection-level and search-level filters
- **Recommendations** - Set up and customize product recommendations
- **Analytics** - Monitor search and discovery performance
- **Predictive search** - Configure autocomplete behavior
- **Managing searchability** - Control which products are visible in search

---

## Relevance for AI Discovery

The Search & Discovery app controls the internal search layer, but also affects how Shopify Catalog data is structured and surfaced. Accurate synonym groups, complete product data, and strong category assignments improve both storefront search and AI platform discoverability.
