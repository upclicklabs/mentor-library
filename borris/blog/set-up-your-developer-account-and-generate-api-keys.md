# Set up your developer account and generate API keys on the Anthropic Console

## Claude Developer Platform

The Anthropic Console has been rebranded to the **Claude Console** and is now located at **platform.claude.com**.

### What This Platform Offers

This is Anthropic's developer platform where you can:
- Access the Claude API
- Manage API keys
- Build applications with Claude
- Configure and manage your development environment

### Available Claude Models

The platform provides access to the latest Claude models:
- **Claude Opus 4.5** (`claude-opus-4-5-20251101`)
- **Claude Sonnet 4.5** (`claude-sonnet-4-5-20250929`)
- **Claude Haiku 4.5** (`claude-haiku-4-5-20251001`)

- **Documentation:** https://docs.claude.com
