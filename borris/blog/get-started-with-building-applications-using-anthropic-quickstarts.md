# Get started with building applications using Anthropic Quickstarts

## Overview

Anthropic Quickstarts is a collection of projects designed to help developers quickly get started building deployable applications using the Claude API. The repository contains ready-to-use foundations that can be easily customized for specific needs.

## Getting Started

To use these quickstarts, you need a Claude API key from console.anthropic.com (free signup available).

### General Setup Steps

1. Clone the repository
2. Navigate to the specific quickstart directory
3. Install required dependencies
4. Set up your Claude API key as an environment variable
5. Run the quickstart application

## Available Quickstarts

### 1. Customer Support Agent

An AI-powered customer support system leveraging Claude's natural language capabilities with knowledge base access.

### 2. Financial Data Analyst

Analyzes financial data via chat with interactive data visualization capabilities.

### 3. Computer Use Demo

Demonstrates Claude's ability to control desktop computers, supporting the latest computer_use tool version with zoom actions.

### 4. Browser Tools API Demo

Complete reference implementation for browser automation, including navigation, DOM inspection, and form manipulation using Playwright.

### 5. Autonomous Coding Agent

Two-agent pattern (initializer + coding agent) that builds complete applications over multiple sessions with progress persistence via git.
