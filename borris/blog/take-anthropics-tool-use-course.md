# Take Anthropic's tool use course

Welcome to Anthropic's comprehensive tool use tutorial. Across six lessons, you will learn everything you need to know to implement tool use successfully in your workflows with Claude. We recommend that you start from the beginning with the tool use overview, as each lesson builds on key concepts taught in previous ones.

## Table of contents

1. Tool use overview (01_tool_use_overview.ipynb)
2. Your first simple tool (02_your_first_simple_tool.ipynb)
3. Forcing JSON with tool use (03_structured_outputs.ipynb)
4. The complete tool use workflow (04_complete_workflow.ipynb)
5. Tool choice (05_tool_choice.ipynb)
6. Building a chatbot with multiple tools (06_chatbot_with_multiple_tools.ipynb)

## Course Overview

This is Anthropic's official comprehensive tutorial on implementing tool use with Claude. The course is structured as a progressive learning path with 6 sequential lessons, each building upon concepts from previous modules.

### Key Topics Covered

- Tool use fundamentals and overview
- Implementing your first tool
- Structured outputs and JSON formatting
- Complete end-to-end tool use workflows
- Tool selection and routing strategies
- Multi-tool chatbot implementations

All lessons are provided as Jupyter notebooks (.ipynb files) in the /tool_use directory of the Anthropic courses repository on GitHub.
