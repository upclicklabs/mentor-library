# Explore Anthropic's tool use cookbooks

This is the tool_use directory from Anthropic's claude-cookbooks repository on GitHub. It is a public repository containing practical examples and cookbooks for using Claude with various features.

## Directory Contents

### Subdirectories

- **memory_demo/** - Demonstration of memory functionality
- **tests/** - Test files
- **utils/** - Utility modules

### Jupyter Notebooks (Educational Examples)

1. **automatic-context-compaction.ipynb** - Context management techniques
2. **calculator_tool.ipynb** - Building a calculator tool
3. **customer_service_agent.ipynb** - Customer service agent implementation
4. **extracting_structured_json.ipynb** - JSON extraction patterns
5. **memory_cookbook.ipynb** - Memory management guide
6. **parallel_tools.ipynb** - Using multiple tools in parallel
7. **programmatic_tool_calling_ptc.ipynb** - Programmatic tool invocation
8. **tool_choice.ipynb** - Tool selection strategies
9. **tool_search_alternate_approaches.ipynb** - Alternative tool search methods
10. **tool_search_with_embeddings.ipynb** - Embedding-based tool discovery
11. **tool_use_with_pydantic.ipynb** - Pydantic integration for tool use
12. **vision_with_tools.ipynb** - Vision capabilities with tools

### Configuration Files

- **.env.example** - Environment variable template
- **requirements.txt** - Python dependencies
- **__init__.py** - Python package initialization

## Purpose

This repository provides practical guides and working examples for implementing tool use with Claude, covering:

- Tool definition and calling patterns
- Structured data extraction
- Agent implementations
- Memory management
- Vision integration
- Advanced patterns like context compaction and parallel execution

The cookbooks are designed to help developers implement Claude's tool-use capabilities in their applications.
