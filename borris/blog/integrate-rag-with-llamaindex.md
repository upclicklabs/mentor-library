# Integrate RAG with LlamaIndex

LlamaIndex is a data framework for LLM-based applications that benefit from context augmentation.

This repository provides cookbooks for building LLM applications using Anthropic's Claude and LlamaIndex together.

## Available Notebooks

1. **Basic RAG With LlamaIndex** (`Basic_RAG_With_LlamaIndex.ipynb`) - Build RAG (Retrieval-Augmented Generation) pipelines with LlamaIndex

2. **Router Query Engine** (`Router_Query_Engine.ipynb`) - Use `RouterQueryEngine` to route user queries to different indices

3. **SubQuestion Query Engine** (`SubQuestion_Query_Engine.ipynb`) - Use `SubQuestionQueryEngine` to answer complex user queries spanning multiple documents

4. **ReAct Agent** (`ReAct_Agent.ipynb`) - Use `ReActAgent` for using Tools and QueryEngine Tools

5. **Multi Document Agents** (`Multi_Document_Agents.ipynb`) - Build an efficient RAG pipeline for a large number of documents

6. **Multi Modal** (`Multi_Modal.ipynb`) - Build Multi-Modal applications using LlamaIndex
