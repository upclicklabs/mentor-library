# Take an in-depth course: Anthropic models using Google Vertex AI

## Course: Claude with Google Cloud's Vertex AI

This comprehensive technical training covers integrating and deploying Claude AI models through Google Cloud's Vertex AI platform. The curriculum focuses on implementing Claude's API capabilities, from fundamental request handling to sophisticated features including tool use, retrieval augmented generation (RAG), and the Model Context Protocol (MCP). The course emphasizes practical implementation patterns, performance optimization, and production-ready workflows for building AI-powered applications.

## Key Learning Outcomes

- Setting up Claude models within Google Cloud's infrastructure
- Managing multi-turn conversations with proper context handling
- Designing and evaluating prompts using systematic testing approaches
- Applying prompt engineering principles including XML structuring and output control
- Building tool-use implementations for external function interaction
- Developing RAG pipelines with text chunking and retrieval techniques
- Utilizing advanced features like vision capabilities and prompt caching
- Implementing the Model Context Protocol for custom tools
- Deploying Anthropic Apps including Claude Code and Computer Use features
- Designing agent-based workflows with parallelization and routing patterns

## Prerequisites

- Proficiency in Python programming
- Experience with Google Cloud Platform
- Understanding of JSON data structures

## Target Audience

Backend developers, full-stack engineers, ML engineers, DevOps professionals, technical architects, and those transitioning from other LLM providers to Claude.
